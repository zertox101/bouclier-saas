```

- alert-high.mp3
- alert-medium.mp3
- alert-info.mp3

Placer dans: frontend/public/sounds/
```

### 3. Tester (15 min)
```
1. Démarrer le frontend
2. Naviguer vers /settings/notifications
3. Tester chaque sévérité
4. Vérifier sons, desktop, toast
5. Modifier configuration
6. Re-tester
```

### 4. Intégrer (30 min)
```typescript
// Dans world-monitor/page.tsx
import { useNotificationContext } from '@/components/notifications/NotificationProvider';

const { notify } = useNotificationContext();

// Notifier sur nouvelle attaque
useEffect(() => {
  if (newAttack.severity === 'CRITICAL') {
    notify({
      id: newAttack.id,
      type: newAttack.type,
      severity: 'CRITICAL',
      message: `Attack from ${newAttack.country}`,
      src_ip: newAttack.src_ip,
      country: newAttack.country,
      timestamp: new Date().toISOString()
    });
  }
}, [newAttack]);
```
