
# CyberDetect / Bouclier Frontend

A futuristic B2B SaaS frontend built with Next.js 14, TailwindCSS, and TypeScript.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Backend API running on port 8005 (or see .env)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

- `src/app/(marketing)`: Public pages (Landing, Pricing, Security)
- `src/app/(auth)`: Authentication pages (Login)
- `src/app/app`: Protected Dashboard application
- `src/components/ui`: Shared Neon/Glass UI components
- `src/lib`: Utilities, API client, and SSE hooks

## 🎨 Theme System

The application uses a custom "Obsidian/Neon" theme extending Tailwind.
- **Glassmorphism**: Usage of backdrop-blur and semi-transparent backgrounds.
- **Neon Accents**: Primary color is Purple (`#7C3AED`), Secondary is Cyan.
- **Dark Mode**: Forced dark mode for the specific cyber-security aesthetic.

## 🔌 API Integration

- **API Client**: `src/lib/api-client.ts` handles fetch with credentials.
- **Realtime**: `src/lib/use-sse.ts` connects to `/api/events/stream`.
- **Mock Mode**: Use `demo@bouclier.io` / `demo` to bypass API authentication if backend is offline.

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: TailwindCSS + Framer Motion (animations)
- **Icons**: Lucide React
- **Charts**: Recharts
