## Android Utilities
* [cSploit](https://github.com/cSploit/android) - Advanced IT security professional toolkit on Android featuring an integrated Metasploit daemon and MITM capabilities.
* [Fing](https://www.fing.com/products/fing-app/) - Network scanning and host enumeration app.
* [MPT](https://github.com/ByteSnipers/mobile-pentest-toolkit) - Mobile Pentest Toolkit.

## Anonymity Tools
* [I2P](https://geti2p.net/) - The Invisible Internet Project.
* [MAT](https://0xacab.org/jvoisin/mat2) - Metadata Anonymization Toolkit.
* [Tor](https://www.torproject.org/) - Free software and onion routed overlay network.

## Cloud Platform Attack Tools
* [Cloud Container Attack Tool (CCAT)](https://rhinosecuritylabs.com/aws/cloud-container-attack-tool/) - Tool for testing security of container environments.
* [CloudHunter](https://github.com/belane/CloudHunter) - Looks for AWS, Azure and Google cloud storage buckets.
* [Cloudsplaining](https://cloudsplaining.readthedocs.io/) - Identifies violations of least privilege in AWS IAM policies.
* [GCPBucketBrute](https://github.com/RhinoSecurityLabs/GCPBucketBrute) - Script to enumerate Google Storage buckets.

## Exfiltration Tools
* [DET](https://github.com/sensepost/DET) - Proof of concept to perform data exfiltration.
* [Iodine](https://code.kryo.se/iodine/) - Tunnel IPv4 data through a DNS server.
* [dnscat2](https://github.com/iagox86/dnscat2) - Encrypted command and control channel over DNS.

## Exploit Development Tools
* [Pwntools](https://github.com/Gallopsled/pwntools) - Rapid exploit development framework.
* [peda](https://github.com/longld/peda) - Python Exploit Development Assistance for GDB.

## Network Tools
* [CrackMapExec](https://github.com/byt3bl33d3r/CrackMapExec) - Swiss army knife for pentesting networks.
* [BetterCAP](https://www.bettercap.org/) - Modular MITM framework.
* [Wireshark](https://www.wireshark.org/) - Widely-used network protocol analyzer.
* [Nmap](https://nmap.org/) - Free security scanner for network exploration.
* [Masscan](https://github.com/robertdavidgraham/masscan) - Fastest Internet port scanner.

## OSINT
* [Maltego](http://www.maltego.com/) - Open sources intelligence and forensics.
* [Shodan](https://www.shodan.io/) - Search engine for Internet-connected devices.
* [theHarvester](https://github.com/laramies/theHarvester) - E-mail, subdomain and people names harvester.

## Reverse Engineering
* [Ghidra](https://www.ghidra-sre.org/) - Suite of free software reverse engineering tools by NSA.
* [IDA Pro](https://www.hex-rays.com/products/ida/) - Multi-processor disassembler and debugger.
* [Radare2](http://rada.re/r/index.html) - Open source reverse engineering framework.

## Web Exploitation
* [Burp Suite](https://portswigger.net/burp/) - Integrated platform for performing security testing of web applications.
* [OWASP ZAP](https://www.zaproxy.org/) - Scriptable HTTP intercepting proxy and fuzzer.
* [SQLmap](http://sqlmap.org/) - Automatic SQL injection tool.
* [WPScan](https://wpscan.org/) - Black box WordPress vulnerability scanner.
* [Nikto](https://github.com/sullo/nikto) - Web server scanner.
* [Gobuster](https://github.com/OJ/gobuster) - Directory/File, DNS and VHost busting tool.

## Windows Utilities
* [Bloodhound](https://github.com/adaptivethreat/Bloodhound/wiki) - Active Directory trust relationship explorer.
* [Empire](https://www.powershellempire.com/) - Pure PowerShell post-exploitation agent.
* [Mimikatz](http://blog.gentilkiwi.com/mimikatz) - Credentials extraction tool.
* [Sysinternals Suite](https://docs.microsoft.com/en-us/sysinternals/downloads/sysinternals-suite) - Troubleshooting Utilities.
