# 🔥 RedHound PRO

Advanced Bug Bounty Framework with AI

## 🚀 Features
- SQLi, XSS, SSRF...
- AI detection
- CVE integration

## ⚙️ Installation
```bash
git clone https://github.com/chahiddoumar/redhound-pro.git
cd redhound-pro
python3 redhound_builder.py
cd redhound
chmod +x build.sh
./build.sh
source venv/bin/activate
python3 app.py
